# Marcel Vibe — guide for future sessions (Claude's notes)

This file is my (Claude's) working memory for helping with the **Marcel Vibe**
YouTube channel. Read it first whenever Marcel asks for help with a new session,
then read the 4 source files in this folder before drafting anything.

## The channel in one line
A beginner DJ/creator films himself DJing at scenic spots (mostly Malta/Gozo,
sometimes Moldova), almost always at **sunset**, in **4K** — filmed on an iPhone
on a stand plus a **drone**, with the audio recorded straight from the DJ
console and synced to the video.

## GOLDEN RULE: only touch the session(s) Marcel is currently working on
NEVER edit, rephrase, or "fix" past sessions (including dashes/typos). The
already-posted entries must stay byte-for-byte in sync with what's live on
YouTube - they are a historical record. Only ever change the exact `#NN`
session(s) Marcel explicitly names in the current request.

## The 4 source files (single source of truth, keep them in sync)
- **Titles.txt** — one title per session (`#NN`).
- **Descriptions.txt** — full YouTube description per session.
- **Tags.txt** — comma-separated YouTube tags per session.
- **Playlists.txt** — `#NN - YYYY-MM-DD Location` header + timestamped tracklist.
  This is the most reliable place to find a session's **date, location and
  tracklist** (Marcel fills it before the description).

## Genre labelling — IMPORTANT, check the tracklist properly
Don't guess the genre from a vibe — read the actual artists/tracks and match them.
Marcel's own labels are the source of truth; confirm with him when unsure.
- The soft, dreamy, emotional "Silk Music" sound — **Shingo Nakamura, A.M.R,
  Sound Quelle, Dokho, Flexible Fire, Approaching Black, RAINE, Jay FM, Stendahl,
  Marsh** — Marcel labels this **Melodic House** (NOT Progressive House, even
  though some stores tag it that way). #21 is the reference example.
- Reserve **Progressive House** for genuinely driving/peak progressive sets.
- Melodic Techno only when there are clearly techno-leaning tracks (e.g. Argy &
  Goom Gum). Use sparingly.
- Other labels seen: Organic House, Deep House, Afro House, Minimal / Deep Tech.

## Title formula
`[Genre] [time/format] Mix at [Location], [Region] | Marcel Vibe [NN]`
- Genre examples: Organic House, Deep House, Melodic House, Progressive House,
  Afro House, Minimal / Deep Tech. Often two joined with `&` or `·`.
- Optional extras: `Sunset` / `Sunrise`, `4K`, `4K Drone`, featured DJ names.
- Lead with whatever makes the session special (e.g. #20 = the only *sunrise*).
- Always end with `| Marcel Vibe [NN]`.

## Description structure (in this order)
1. Short, honest, first-person story (3–6 sentences). This is the heart of it.
2. Optional special call-out (e.g. "first time filming with the drone", a
   timestamp of a sweet moment, a parrot attacking the drone).
3. `🙏` thanks to whoever filmed / helped with the drone.
4. Metadata block, each on its own line:
   `📍 Location:` / `🕐 Time:` / `🎥 Shot in 4K (...)` / `🎵 Genres:`
   (DJs line `🎧 DJs:` when there are guests).
5. `🎶 Tracklist` with timestamps (recent style uses `HH:MM:SS`).
6. A line of **exactly 3 hashtags** (standard from #20 on). YouTube only shows
   the first 3 above the title and ignores everything past 15, and the heavy
   discovery work lives in Tags.txt — so keep it to 3 clean, series-relevant
   tags and skip the long block (it reads spammy and dilutes the human tone).
   Start with `#MarcelVibe`, then 2 that fit the session, e.g.
   `#MarcelVibe #SunriseMix #MelodicHouse`.

## Voice / tone rules
- First person, warm, humble, conversational. Real English of a non-native
  speaker — polish it, but DO NOT make it sound corporate or AI-perfect.
- Honesty is the brand: Marcel openly shares mistakes (overexposed footage,
  dropped phone in #14, amateur cameraman in #15, "this isn't perfect" in #19).
  Keep that. It's what makes it feel human.
- Lightly rephrase the raw text he provides; keep his meaning and his humility.
- Use emojis the way the existing descriptions do, sparingly, as section markers.
- PUNCTUATION: Marcel does NOT use em dashes (—) or en dashes (–). Always use a
  plain hyphen "-" (e.g. "set - and it wasn't easy", "DUALIS - Avalon"). Never
  output — or – anywhere in titles, descriptions, or tags.

## Tags style (Tags.txt)
Comma-separated, no spaces needed between items. Always include MANY branding
variants so search catches them all:
`Marcel Vibe,marcel vibe [NN],MarcelVibe[NN],marcelvibe[NN],MarcelVibe [NN],Marcel Vibe [NN]`
Then: genres, location (+ region/country), "malta sunset/sunrise", "4k drone",
and the notable artists/DJs in the set.

## Workflow when Marcel posts a new session
1. Read all 4 files + this README.
2. Get date/location/tracklist from Playlists.txt for that `#NN`.
3. Infer genre from the tracklist artists.
4. Draft TITLE (formula above) — lead with what's special.
5. Rephrase his raw description text → keep human + honest.
6. Draft TAGS.
7. With his OK, fill the `#NN` slot in Titles.txt, Descriptions.txt, Tags.txt.
8. Ask when unsure (genre label, guest DJ spelling, whether to credit handles).
